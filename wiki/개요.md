# NemoEngine 프로젝트 개요

**NemoEngine**은 SillyTavern(실리태번) 및 RoleCall 계열 AI 롤플레이 / 협업 소설 워크플로우를 위한 **대형 모듈형 프리셋 모음**입니다.

## 프리셋이란?

AI에게 "어떻게 서술할지, 캐릭터를 어떻게 다룰지, 유저에게 얼마나 자율성을 줄지, 어떤 문체로 쓸지, 어떤 부가 시스템을 켤지"를 지시하는 **명령어 묶음**입니다. JSON 파일 하나를 SillyTavern에 임포트하면 되고, 같은 내용이 `Prompts/` 폴더에 읽기 좋은 마크다운 조각으로 분해되어 있어 검사·수정·검색이 가능합니다. → 자세한 용어는 [[용어집]] 참고.

## 세 가지 프리셋 가족

| 프리셋 | 성격 | 추천 대상 |
|---|---|---|
| [[네모엔진]] (Nemo Engine) | 메인 큐레이션 프리셋. 성숙하고 안정적 | 처음 쓰는 사람, 일상적 롤플레이 |
| [[네모넷]] (NemoNet) | 최대 규모 실험작. 스위치가 가장 많음 | 모델 실험, 극한 커스터마이징 |
| [[아틀리에]] (Atelier) | 취향 슬라이더 기반 가이드형 프리셋 | "어떤 이야기 원해?"식 간편 설정 |

셋 다 같은 설계 사상을 공유합니다: [[코어팩-변수-시스템]]으로 모순 없는 조립, [[설계-철학]](유저 주도권 + NPC 자율성 + 마음이론), [[벡스]]라는 화자 페르소나, [[CoT-추론-시스템]], [[트래커]] 같은 선택적 UI 시스템.

## 현재 규모 (2026-07 기준)

| 가족 | 임포트 JSON | 프롬프트 파일 수 |
|---|---|---:|
| Nemo Engine | `Nemo Engine v10.json` | 323 |
| NemoNet | `Nemo Net 2.0.json` | 511 |
| Atelier | `Atelier 2.1.json` | 96 |

프롬프트 대부분은 **의도적으로 비활성** 상태입니다. 죽은 코드가 아니라 "필요할 때 켜는 옵션"입니다 (장르 모듈, Vex 성격, 트래커, 성인 콘텐츠 토글 등).

## 관련 프로젝트

- **RoleCall** (rolecallstudios.com) — 인터랙티브 픽션 채팅 프론트엔드 / 캐릭터 스튜디오
- **Plotlight** (plotlightstudios.com) — 롤플레이 콘텐츠 공유·탐색 플랫폼

장기 비전: 이 프리셋들이 떠도는 프롬프트 파일이 아니라, 인터랙티브 픽션을 만들고·공유하고·플레이하는 생태계의 일부가 되는 것.

## 저장소 구조

```
Nemo Engine/   메인 프리셋 (JSON + Prompts/ + Archive/ + Archived Prompts/)
NemoNet/       실험 프리셋 (동일 구조)
Atelier/       취향 기반 프리셋 (동일 구조)
Templates/     독립 실험 프롬프트 모음
```

버전별 변천사는 [[버전-히스토리]] 참고. 직접 모듈을 만들고 싶다면 [[커스터마이징-가이드]] 참고.
