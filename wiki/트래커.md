# 트래커 (Trackers)

응답에 **눈에 보이는 상태 표시**를 추가하는 선택 모듈군입니다. 퀘스트, 위치, 관계 진행도, 선택지, 게임 UI 같은 것을 산문 옆에 렌더링합니다.

## 렌더 모드

트래커 토글을 켜고 렌더 모드를 **하나** 선택: HTML/CSS · ASCII · Regex 렌더 · 일반 마크다운. HTML 출력의 품질 검증은 HTML Design / Marker Check CoT 스텝이 담당 → [[CoT-추론-시스템]].

## 네모엔진 트래커 (Trackers 카테고리 32개)

Status Board(상태 보드), Quest Journal(퀘스트 저널), Location Board, CYOA(선택지 제시), Dating Sim(연애 시뮬 UI), Manga Panels / Webtoon Panels(만화·웹툰 패널 연출), Immersive World HTML, Auto Image Gen(자동 이미지 프롬프트), Gacha System, Knowledge Log, Fandom Reaction, Scroll News & Lore 등.

## 네모넷 v2 추가 트래커 (15종)

Danmaku Comments(니코니코식 탄막), Blood Bond, Corruption Meter, Option Bar, Game Interface, Summary System, Word Count, Objective Tracker, Story Tracker, Story Ledger, Scene Status, Heart Chart, Visual Inventory, Texting Style(문자 메시지 연출), Parallel Events → [[네모넷]]

## 사용 원칙

- **존재한다고 켜지 말 것.** 트래커는 출력 토큰을 먹고, 장면에 불필요하면 산문을 방해한다
- 렌더 모드 하나 + 구체적 트래커 한두 개가 적정
- 저사양 모델에서는 대형 트래커 출력 회피

기술적으로는 트래커 변수를 채우면 코어팩이 정해진 위치에 렌더하는 구조 → [[코어팩-변수-시스템]]. 새 트래커 제작은 [[커스터마이징-가이드]] 참고.
