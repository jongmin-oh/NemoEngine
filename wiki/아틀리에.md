# 아틀리에 (Atelier)

취향 기반 가이드형 프리셋. 현재 버전 **2.1** (`Atelier/Atelier 2.1.json`, 프롬프트 96개). 거대한 모듈 카탈로그 대신 **"어떤 경험을 원하세요?"라는 질문에 답하는 방식**으로 이야기를 설정합니다.

## 작동 방식: 인터뷰 슬라이더

프롬프트 파일명을 보면 구조가 보입니다. 각 취향 항목마다 헤더(질문) + 상/중/하 선택지가 있고, 하나만 켭니다:

```
"⚡ 판돈(stakes)이 얼마나 높아야 해?"
  💥 More Stakes  /  ⚖️ Stakes: Meh  /  🌿 Less Stakes
"🎯 세계가 널 어떻게 대해?"
  🥊 Make Me Believe It  /  ⚖️ Honest World  /  🌤️ Wind At Your Back
```

## 취향 다이얼 목록

- 판돈과 긴장감 (stakes)
- 세계 편향 (world bias — 2.0에서 추가)
- 성적 콘텐츠 수위 / 폭력 강도 / 로맨스 수준 / 유머
- 응답 길이 / 감각 묘사 밀도 / 무드와 톤
- 시점(POV: 유저 시점, 캐릭터 시점, 작가 시점)
- 스토리 주도자 (유저 주도 vs AI 주도) → [[설계-철학]]
- 대화 vs 산문 비율 / 세계관 깊이 / 캐릭터 성장
- 세계 논리 (Grounded 현실적 / Genre Logic 장르 논리 / Anime 애니메 논리)
- 비주얼 포맷 (Plain Text / Immersive Formatting / Character Colours)

## 코어 구성요소

- **The Premise** — 프리셋 도입부
- **Writing Style Library** — 대형 문체 라이브러리
- **Character Anchor** — 캐릭터 일관성 고정
- **Dynamic Progression / Living World** — 이야기·세계의 능동적 진행
- **The Prose Contract** — 안티슬롭(anti-slop) 규칙 → [[용어집]]
- **Atelier Core Pack** — 선택된 다이얼들을 조립하는 코어팩 → [[코어팩-변수-시스템]]
- **Databank / Settings Reminder** — 장기 기억 보조
- 선택적 CoT 플래닝 프롬프트 → [[CoT-추론-시스템]]

## 언제 쓰나

모듈 이름을 몰라도 되는 프리셋을 원할 때. "판타지 로맨스, 판돈 낮게, 유머 적당히, 유저 주도"처럼 취향만 고르면 됩니다. 세밀한 제어가 필요해지면 [[네모엔진]]으로, 극한 실험은 [[네모넷]]으로.
