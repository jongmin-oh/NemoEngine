# NemoEngine 위키 색인

한국어로 정리한 NemoEngine 프로젝트 지식 베이스. 시작점은 [[개요]].

## 개요

- [[개요]] — 프로젝트 전체 소개, 세 프리셋 가족 비교, 저장소 구조

## 프리셋

- [[네모엔진]] — 메인 프리셋 (v10, 323 프롬프트). 첫 세팅 권장안과 상황별 조합
- [[네모넷]] — 최대 규모 실험 프리셋 (2.0, 511 프롬프트). v10 spine 재구축, 렌즈, 랜덤/픽 모드
- [[아틀리에]] — 취향 슬라이더 프리셋 (2.1, 96 프롬프트). 인터뷰식 다이얼 구조

## 핵심 개념

- [[코어팩-변수-시스템]] — setvar/getvar 조립 아키텍처. **기술적으로 가장 중요한 페이지**
- [[벡스]] — 내레이터 정체성(저자의 렌즈)과 20+ 성격 모드
- [[CoT-추론-시스템]] — Main/Fast CoT와 모듈형 사고 스텝
- [[트래커]] — 시각적 상태 표시 모듈군과 렌더 모드
- [[설계-철학]] — 유저 주도권 · NPC 자율성 · 마음이론 · 안티슬롭
